# KijaniKiosk DevOps

Engineering documentation and operational runbooks for the KijaniKiosk platform.

This repository tracks DevOps practices, delivery pipeline improvements, and
incident response procedures used by the KijaniKiosk engineering team.

## Engineering Standards
All changes go through pull requests. No direct commits to main or develop.

## On-Call Policy
The engineer who deploys owns the incident if it fires within 24 hours.
